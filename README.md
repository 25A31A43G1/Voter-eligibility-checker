# Smart Voter Eligibility System

A C++ mini project that checks whether a person is eligible to vote based on age and voter ID validation. The system collects user details such as name, age, gender, country, and voter ID, then verifies eligibility conditions and displays the result.

## Features
- Age-based eligibility check
- Voter ID validation
- Multiple user support
- User-friendly console interface
- Beginner-friendly C++ project

## Technologies Used
- C++
- Conditional Statements
- Loops
- String Handling

## Purpose
This project was developed to improve programming logic and understand real-world application development using C++.
